### Exercise 5: Jenkins-Pipeline in a Codefile
###  Prerequisite
1. A Jenkins controller is running on AWS.
2. Previous exercise has been implemented successfully.
### Tasks
1. Copy pipeline of the previous exercise in a file (i.e. Jenkinsfile1) and push it your public repo.
2. Set up an SCM (Source code management)-pipeline in order to execute pipeline steps stored in the Jenkins file.
3. Build and check the new pipeline.
